## Features

<CardGrid>
  <Card title="Clean Artist Browsing" icon="star">
    No more dozens of "Artist feat. X" cluttering your iPod. The clean artists feature tidies up
    artist names on the fly — your source files stay untouched.
  </Card>
  <Card title="Automatic Transcoding" icon="setting">
    Music and video transcoded on the fly with device-aware quality presets. Keep your high-quality
    collection — podkit optimises for your iPod.
  </Card>
  <Card title="Non-Destructive" icon="approve-check-circle">
    Your music collection is never modified. Transforms, transcoding, and format conversion only
    affect what lands on the iPod.
  </Card>
  <Card title="Incremental Sync" icon="rocket">
    Only new, changed, or removed tracks are touched. Re-syncing a large library takes seconds, not
    hours.
  </Card>
  <Card title="Multiple Sources" icon="document">
    Sync from local directories or a Subsonic-compatible server like Navidrome. Define multiple
    collections and sync independently.
  </Card>
  <Card title="Every Classic iPod" icon="open-book">
    Stock firmware on all classic iPod models, including modded iPods with iFlash adapters.
  </Card>
</CardGrid>

## Install

```bash
brew install jvgomg/podkit/podkit
```

Other install methods are available — see the [Installation guide](/getting-started/installation).

## Quick Start

```bash
# Add your music (or Navidrome — see docs)
podkit collection add -t music -c main --path ~/Music/library

# Add a video collection
podkit collection add -t video -c movies --path ~/Movies

# Register your iPod
podkit device add -d myipod

# Mount, sync, and eject
podkit device mount
podkit sync --dry-run
podkit sync
podkit device eject

# See all available commands
podkit --help
```

## Supported Devices

podkit works with iPods that use USB Mass Storage mode with the iTunesDB database format:

- **iPod Classic** — all generations
- **iPod Video** — 5th and 5.5th generation
- **iPod Nano** — 1st through 5th generation
- **iPod Mini** — 1st and 2nd generation
- **iPod Shuffle** — 1st and 2nd generation

Modded iPods with iFlash SD card adapters are fully supported. See the [complete compatibility list](/devices/supported-devices).

**Not supported:** iOS devices (iPod Touch, iPhone, iPad) use a different sync protocol.
