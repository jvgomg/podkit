## Description

Pure TypeScript iPod database implementation replacing @podkit/libgpod-node. Implements the 24 methods used by podkit-core, all hash algorithms (hash58/72/AB), full device support including big-endian, and all artwork pixel formats. See design document doc-003 for full details.
