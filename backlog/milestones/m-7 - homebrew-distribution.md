## Description

Enable users to install podkit via Homebrew on macOS and Linux. Covers changesets versioning, CI release pipeline producing standalone compiled binaries, a homebrew-podkit tap repo with auto-updating formula, and updated documentation. When complete: `brew install jvgomg/podkit/podkit` gives users a working CLI with no runtime dependencies beyond FFmpeg.
