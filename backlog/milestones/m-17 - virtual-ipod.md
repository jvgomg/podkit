## Description

A virtual iPod application for demonstrating podkit. Includes: pure TS read-only iTunesDB parser (shared with m-8), iPod web UI/firmware as a reusable web component, browser and remote storage adapters, a Lima VM backend with USB gadget emulation for real podkit device compatibility, and a standalone macOS Electron app. The read-only parser work feeds directly into m-8 (ipod-db Core) — those tasks serve both milestones.
